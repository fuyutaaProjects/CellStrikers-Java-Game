# Comment générer les fichiers documentation des classes:

Pour cela dans le répertoire ou se trouve ce fichier il y a deux fichiers: **docGen.bat** et **docGen.sh**. <br>
Ce sont deux script, le premier est à exécuter si on est sous *windows* et le deuxième si on est sous *linux* ou *macOs*. <br>
Après l'avoir lancer, tous les fichiers de documentations seront dans le répertoire **doc**. Pour consulter cela, il faut aller
dans ce répertoire et ouvrir avec son navigateur le fichier **index.html**.
