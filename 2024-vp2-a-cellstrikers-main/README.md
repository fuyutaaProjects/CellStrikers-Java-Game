## Lancement du projet
Le lancement du projet peut se réaliser via la compilation dans un IDE du fichier **GAME** c'est le fichier qui contient le main.
<br>Sinon le lancement peut se faire via les fichiers start, on peut retrouver deux fichiers start : start.bat et start.sh, ils permettent de lancer le projet sans IDE
